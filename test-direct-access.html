브라우저 캐시 우회 테스트

이 페이지는 브라우저 캐시 문제를 해결하기 위한 직접 접속 링크들입니다.

다음 링크들을 시도해보세요:

  • 일반 접속
  • 캐시 무시 1
  • 버전 파라미터
  • 타임스탬프 파라미터

강제 새로고침 방법:

  • Chrome/Edge: Ctrl+Shift+R 또는 Ctrl+F5
  • Firefox: Ctrl+Shift+R 또는 Ctrl+F5
  • Safari: Cmd+Shift+R
  • 개발자 도구: F12 → Network 탭 → "Disable cache" 체크

확인사항:

정상적으로 적용되었다면 다음이 보여야 합니다:

  1. 📹 세션 기록 섹션
  2. 🧪 상호작용 테스트 섹션 (새로 추가된 부분)
  3. 📋 기록된 세션 섹션
  4. 🎥 세션 리플레이 섹션
